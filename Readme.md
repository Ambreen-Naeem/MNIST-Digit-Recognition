# 🔢 MNIST Digit Recognition Using Machine Learning

## Overview

This project implements a handwritten digit recognition system using the MNIST dataset and a Random Forest Classifier. The model is trained to identify handwritten digits from 0 to 9 and is integrated with a Streamlit web application for real-time digit prediction.

The application allows users to upload handwritten digit images and receive instant predictions through an interactive web interface.

---

## Features

* Handwritten digit classification (0–9)
* Data preprocessing and normalization
* Random Forest machine learning model
* Dataset augmentation using inverted images
* Model evaluation using multiple metrics
* Model saving using Pickle
* Interactive Streamlit frontend
* Automatic background handling for uploaded images

---

## Dataset

The project uses the MNIST Handwritten Digits Dataset.

### Dataset Information

* Total Images: 42,000
* Image Size: 28 × 28 Pixels
* Color Format: Grayscale
* Classes: 10 (Digits 0–9)

## Dataset

Download the MNIST dataset from Kaggle and place `train.csv` in the project root directory before training the model.

Dataset: https://www.kaggle.com/competitions/digit-recognizer/data

## Installation

Clone the repository:

```bash
git clone <repository-link>
cd MNIST-Digit-Recognition
```
Install required libraries:

```bash
pip install -r requirements.txt
```

---

## Training the Model

Run:

```bash
python mnist_digit_recognition.py
```

This will:

* Train the model
* Evaluate performance
* Save the trained model as `mnist_model.pkl`

---

## Running the Streamlit Application

Run:

```bash
streamlit run app.py
```

Open the local URL generated by Streamlit in your browser.
